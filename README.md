# alanote
An AI powered audio-note taking application for students using your choice of Ollama, Claude, ChatGPT, and Grok.

Simply hit record, and when you are ready, stop recording, and click 'Generate notes'. The AI you select will generate a personalized, to-the-point note sheet that is easy to follow in case you are studying or completing homework. Here is how to install:

1. Before you do anything, please make sure you have an Android (computer or phone), Linux*, Mac, or Windows PC. IOS NOT SUPPORTED AS OF NOW-WILL ADD ASAP.
2. Either download latest tarball, apk, or executable from the releases page.
3. Select your preferred AI provider (Ollama, Claude, ChatGPT, or Grok)
4. Start recording!

For those wishing to build from source (PC/Laptop only):
1. Install python
2. Install requirements
   - pip(3) install -r requirements.txt
   - for some versions of python, you will have to manually install the Tkinter library (e.g. sudo apt install python3-tk for Debian-based distros)
3. To run, simply type 'python(3) main.py in the project folder in your command line.
